# 3D Model Credits

The real 3D models used by Grand Theft Moore are all **CC0 1.0 (Public Domain
Dedication)** — no attribution is legally required, but it is listed here as
good practice. Both were downloaded from [Poly Pizza](https://poly.pizza),
which hosts the source creators' CC0 assets.

| File | Model | Author | Source | License | Size |
|------|-------|--------|--------|---------|------|
| `car.glb` | Car | Quaternius | https://poly.pizza/m/unqqkULtRU | CC0 1.0 | 174 KB |
| `character.glb` | Fitness Character | iPoly3D | https://poly.pizza/m/eMOTyGEAxj | CC0 1.0 | 199 KB |

Direct download URLs (Poly Pizza CDN):
- Car: https://static.poly.pizza/d35173c8-6078-4367-8f87-b1f2599f0bb7.glb
- Fitness Character: https://static.poly.pizza/ca6d7f88-fb9e-4c6c-8d6e-acfce081a21d.glb

Both models are self-contained binary glTF (`.glb`) with vertex colours baked
from per-material `baseColorFactor` (no external textures). They are used as
static meshes — any skinning/animation data is ignored by the loader.
